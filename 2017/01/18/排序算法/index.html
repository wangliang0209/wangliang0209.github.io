排序

  • 插入排序
    （时间复杂度 最坏情况 1 + 2 + … + n-1 = n ^2/ 2 即 O（n^2） 最好的情况 O（n） 已经有序，每个只比一次空间复杂度 O（1））

结果演示
[5, 10, 7, 8, 6]
[5, 7, 10, 8, 6]
[5, 7, 8, 10, 6]
[5, 6, 7, 8, 10]

  • 选择排序
    （时间复杂度 最好最坏都是O（n^2）空间复杂度 O（1））

结果演示
[5, 8, 7, 6, 10]
[5, 6, 7, 8, 10]
[5, 6, 7, 8, 10]
[5, 6, 7, 8, 10]
[5, 6, 7, 8, 10]

  • 冒泡排序
    （时间复杂度 最坏情况 1 + 2 + … + n-1 = n ^2/ 2 即 O（n^2）
    空间复杂度 O（1））

  • 快速排序
    （时间复杂度 最坏情况 1 + 2 + … + n-1 = n ^2/ 2 即 O（n^2）最好 O(nlgn)
    空间复杂度O(log2n)~O(n)）
